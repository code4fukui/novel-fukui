# novel-fukui

English README is here: [README.md](README.md)

日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

福井県を舞台とした日本語の短編小説や小説のコレクション。

## 機能
- テキストと対話形式の視覚小説の2つのストーリーがあります:
  - 「村崎紫式部の越前宮殿での一日」
  - 「越前市の黄金の宝物伝説」
- `egadv`ライブラリを使用して作成された対話型の視覚小説

## egadv
ビジュアルノベル(アドベンチャーゲーム)の作成入門
