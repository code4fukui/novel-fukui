# novel-fukui
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

福井県を舞台とした日本語の短編小説や小説のコレクションです。

## 機能
- 「蓮月紫の一日 - 越前宮廷で-」と「越前城の黄金の宝物の伝説」の2つのストーリーがテキストとインタラクティブ小説形式で利用可能:
  - 「蓮月紫の一日 - 越前宮廷で-」
  - 「越前城の黄金の宝物の伝説」
- `egadv` ライブラリを使用して作成されたインタラクティブ小説

## egadv
冒険ゲーム（ビジュアルノベル）の作成入門
