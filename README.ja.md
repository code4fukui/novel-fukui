# novel-fukui
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

福井県を舞台とした日本語の物語やノベルの集まりです。

## 特徴
- テキストと対話型のビジュアルノベルの2つのストーリーを利用可能:
  - 「武蔵野紫の越前宮殿での一日」
  - 「越前市の黄金の宝物伝説」
- `egadv`ライブラリーを使って作成された対話型ビジュアルノベル

## egadv
ビジュアルノベル(アドベンチャーゲーム)の作成入門  
https://github.com/code4fukui/egadv
