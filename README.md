# novel-fukui

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Interactive historical novels set in Fukui, Japan. These stories are written in Markdown and designed to be played in a web browser using the [egadv](https://github.com/code4fukui/egadv) visual novel engine.

## The Novels

### 紫式部の国府での一日 (A Day in the Life of Murasaki Shikibu)
Follow Murasaki Shikibu on a reflective walk through a park in Echizen, recalling her journey from the capital and finding inspiration for her future tales. This story incorporates photography of local scenery.

- **[Play in Browser (egadv)](https://code4fukui.github.io/egadv/?url=https://code4fukui.github.io/novel-fukui/紫式部の国府での一日.md)**
- **[View Script (Markdown)](紫式部の国府での一日.md)**

### 越前市黄金伝説 (The Legend of Echizen City Gold)
Join Mei, a history-loving student, and a talking stork on a treasure hunt to uncover golden artifacts hidden throughout Echizen City based on local history and legends.

- **[Play in Browser (egadv)](https://code4fukui.github.io/egadv/?url=https://code4fukui.github.io/novel-fukui/越前市黄金伝説.md)**
- **[View Script (Markdown)](越前市黄金伝説.md)**

## How It Works
This project uses [egadv](https://github.com/code4fukui/egadv), a simple engine for creating adventure games and visual novels.
- **Markdown-Based:** Each story is a single Markdown file.
- **Branching Narratives:** `## Scene Title` creates a new scene, and `[Choice Text](#Scene-Title)` creates a player choice.
- **Simple to Create:** Anyone can write their own interactive story with basic Markdown knowledge.
